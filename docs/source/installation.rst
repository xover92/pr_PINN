Installation guide
==================

Python version supported : 3.5 | 3.6 | 3.7 | 3.8 | 3.9 | 3.10 | 3.11 | 3.12 | 3.13

The easiest way to the get the `pr_PINN` package in `Python` is via `pip` installation

.. code-block:: bash

  python -m pip install .

The `Python` installation for *developers* is executed using the setup_ script.

.. code-block:: bash

  python -m pip install --editable . --user

.. _setup: https://github.com/xover92/pr_PINN/blob/main/setup.py
